## What you get

- **Auto-fix** — Automatically fix trailing whitespace, style issues, and more
- **Batch scanning** — Scan multiple repositories at once
- **Custom rules** — Define your own review rules
- **Priority support** — Direct email support

## How it works

1. Install reviewkit: `pip install reviewkit`
2. Activate your license: `reviewkit pro RK-XXXX-XXXX-XXXX-XXXX`
3. Use Pro features: `reviewkit fix ./src`

## Why reviewkit?

- **100% local** — Your code never leaves your machine
- **25 built-in rules** — Security, performance, code quality
- **GitHub integration** — Review PRs directly from CLI
- **Multiple output formats** — Text, JSON, Markdown
- **CI-friendly** — Exit codes for blocking critical issues

Python, JavaScript, TypeScript, Go, and more. Works on any codebase.
